## Getting Started
### Deploy already on vercel

### Deploy on local machine
First, install the libiaries:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
